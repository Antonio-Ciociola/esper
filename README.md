# Esper

An esp32 based eurorack module.

![alt](./plots/esper_v1.png)

## Specifications

- 8HP width
- 4 CV inputs
- 2 Audio Outputs
- Oled Display with LVGL GUI
- Rotary Encoder with Push Button
- Esp32s3 has 2 cores, Bluetooth and Wifi support
